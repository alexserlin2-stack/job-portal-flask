# job-portal-flask
 Overview

This is a simple Job Portal Web Application built using Python Flask and SQLite.
It allows admins to post jobs and users to view and apply for jobs through a web interface.

 Features

 Post Job (Admin)
 View Available Jobs
 Apply for Jobs
 View Applications
 Data stored using SQLite database

 Technologies Used

* Python
* Flask
* SQLite
* HTML (Frontend)

Project Structure

job-portal-flask/
│── app.py
│── jobs.db
│── templates/
│     ├── index.html
│     ├── jobs.html
│     └── applications.html

 How to Run

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000

 Usage

* Admin can post job titles
* Users can view jobs and apply
* Applications are stored and can be viewed
